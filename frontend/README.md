# Soil Data Management Platform - Frontend

## Setup

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

## Features

- **Data View**: Browse and explore soil datasets
- **Statistics**: View statistical analysis and missing values
- **Train Model**: Train ML models for soil property predictions
- **Predict**: Make predictions using trained models

## Backend Connection

Make sure the backend server is running on `http://localhost:5000` before using the frontend.

## Usage

1. Start backend: `cd backend && python app.py`
2. Start frontend: `cd frontend && npm run dev`
3. Open browser to `http://localhost:5173`
